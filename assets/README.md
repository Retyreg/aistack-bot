# assets/

Бинарные материалы, которые бот отдаёт пользователям (попадают на сервер
через `git`, поэтому коммитятся в репозиторий).

## AIstack-competitor-analysis-prompt.pdf

Лид-магнит с вебинара 10.06 — промт «Конкурентный анализ v2.0».
Отдаётся в `handlers/start.py` на вебинар-источниках (`is_webinar_source`).

**Файл обязателен.** Положи `AIstack-competitor-analysis-prompt.pdf` в эту
папку и закоммить — иначе `/start` с вебинар-источника упадёт на
`answer_document` после отправки intro, и лид-магнит зациклится.
